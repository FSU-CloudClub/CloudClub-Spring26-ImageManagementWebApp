# AWS Cloud Club Project

## 📂 File Hierarchy
* `/src`: Main application source code.
* `/docs`: Technical documentation and diagrams.
* `/infrastructure`: Terraform or CloudFormation templates for AWS.
* `/.github` : Helpful documentats on various topics.

## 💾 Contribution Rules 
To keep the `main` branch stable, please follow these steps:
1. **Branching:** Never push directly to `main`. Create a branch: `git checkout -b feature/your-feature-name`.
2. **Pull Requests:** Always open a PR to merge into `main`.
3. **Review:** Wait for at least one "LGTM" (Looks Good To Me) from a teammate before hitting merge.
4. **Squash & Merge:** Use the "Squash and merge" option to keep the history clean.

## ‼️ Security
Do **not** commit any `.pem` files, AWS Access Keys, or `.env` files. Ensure they are listed in the `.gitignore`.
